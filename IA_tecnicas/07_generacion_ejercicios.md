# La Necesidad

La práctica deliberada requiere:
* Ejercicios que desafíen lo que necesitás mejorar
* Variedad y progresión
* Casos de prueba para verificar

# ¿Cuándo usarlo?

* Terminaste los ejercicios disponibles
* Necesitás más práctica
* Querés ejercicios para debilidades específicas
* Preparación para exámenes

<!-- _class: code -->

# Plantilla Principal

```text
Generame 5 ejercicios sobre [tema] con 
estas características:

- Dificultad: principiante/intermedio/avanzado
- Que practiquen específicamente: [concepto]
- Con ejemplos de entrada y salida esperada
- Sin darme las soluciones todavía
```

---

<!-- _class: code -->

# Variante: Después de Resolver

```text
Resolví este ejercicio:

[descripción y tu solución]

Ahora quiero profundizar. ¿Podrías:
1. Sugerirme variaciones del problema
2. Proponerme un ejercicio más difícil 
   que practique las mismas ideas
3. Darme casos de prueba especiales
```

---

# Progresión Típica

1. Ejercicios básicos del tema
2. Variaciones con casos límite
3. Combinación con otros conceptos
4. Problemas más desafiantes
