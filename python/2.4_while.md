<!-- _header: '¿Qué es un lazo?' -->

# Repetir acciones

**Hasta ahora:**
* Código se ejecuta una vez
* Secuencial: línea por línea

**Con lazos:**
* Repetir código múltiples veces
* Sin copiar y pegar
* Código más eficiente

<!--
NOTAS DEL ORADOR:
- El concepto de iteración.
- Evitar el código repetitivo (DRY: Don't Repeat Yourself).
- La computadora es excelente para hacer lo mismo millones de veces sin cansarse ni equivocarse.
-->

<!-- _header: 'while' -->

# Lazo while: "Mientras que..."

**Sintaxis:**
```python
while condicion:
    # Código que se repite
    # mientras condicion sea True
```

**Se repite mientras:**
* La condición es True
* Cuando se vuelve False, para

<!--
NOTAS DEL ORADOR:
- Es como un `if` que, al terminar, vuelve a preguntar.
- Si la condición es False de entrada, el código nunca se ejecuta.
-->

---

<!-- _header: 'Ejemplo simple' -->

# Contar del 1 al 5

```python
contador = 1

while contador <= 5:
    print(f"Contador: {contador}")
    contador = contador + 1

print("Fin del lazo")
```

**Salida:**
```
Contador: 1
Contador: 2
Contador: 3
Contador: 4
Contador: 5
Fin del lazo
```

<!--
NOTAS DEL ORADOR:
- Rastrear el valor de `contador` en cada vuelta.
- Mostrar cómo el `print` final está fuera del lazo (no indentado).
-->

---

<!-- _header: 'Flujo de ejecución' -->

# ¿Cómo funciona?

**Paso a paso:**
1. contador = 1
2. ¿1 <= 5? Sí → Ejecuta bloque
3. Imprime "Contador: 1"
4. contador = 2
5. ¿2 <= 5? Sí → Ejecuta bloque
6. ...continúa...
7. ¿6 <= 5? No → Sale del lazo

<!--
NOTAS DEL ORADOR:
- "Trace" o seguimiento manual.
- Es vital entender que la condición se chequea AL PRINCIPIO de cada vuelta.
-->

---

<!-- _header: 'Componentes clave' -->

# Tres partes esenciales

**1. Inicialización:**
```python
contador = 1  # Antes del while
```

**2. Condición:**
```python
while contador <= 5:  # Se evalúa cada vez
```

**3. Actualización:**
```python
contador = contador + 1  # Dentro del while
```

**Sin la 3, lazo infinito**

<!--
NOTAS DEL ORADOR:
- El "Trípode" del while. Si falta una pata, se cae.
- Si no inicializo -> Error (variable no existe).
- Si no actualizo -> Lazo infinito.
-->

# <!-- fit --> ⚠️ Lazos Infinitos
## El error más común

<!--
NOTAS DEL ORADOR:
- Momento de advertencia seria.
-->

---

<!-- _header: 'Lazo infinito' -->

# ¡Cuidado!

**❌ Problema:**
```python
contador = 1
while contador <= 5:
    print(contador)
    # ¡Olvidamos incrementar!
```

**Resultado:**
* Imprime "1" infinitamente
* contador siempre vale 1
* Nunca llega a 6
* El programa se cuelga

<!--
NOTAS DEL ORADOR:
- Explicar qué hacer si pasa: Ctrl+C en la terminal o el botón de stop en Jupyter.
- No rompe la computadora, pero consume CPU.
-->

---

<!-- _header: 'Lazo infinito' -->

# Cómo evitarlos

**✅ Solución:**
```python
contador = 1
while contador <= 5:
    print(contador)
    contador += 1  # ¡No olvidar!
```

**Verificá siempre:**
* ¿La condición cambia?
* ¿Eventualmente será False?
* ¿Hay forma de salir?

<!--
NOTAS DEL ORADOR:
- Programación defensiva.
- Pensar: "¿En qué momento esto se detiene?".
-->

---

<!-- _header: 'break' -->

# Salir del lazo con break

**break = terminar lazo inmediatamente**

```python
contador = 1
while True:  # Lazo infinito a propósito
    print(contador)
    if contador == 5:
        break  # Sale del lazo
    contador += 1

print("Terminó")
```

**Útil para menús y validaciones**

<!--
NOTAS DEL ORADOR:
- `while True` parece peligroso pero con `break` es una herramienta poderosa.
- Es la "salida de emergencia".
-->

---

<!-- _header: 'continue' -->

# Saltar a siguiente iteración

**continue = siguiente vuelta inmediata**

```python
contador = 0
while contador < 10:
    contador += 1
    if contador % 2 == 0:
        continue  # Saltar pares
    print(contador)  # Solo impares
```

**Salida:** 1, 3, 5, 7, 9

<!--
NOTAS DEL ORADOR:
- Diferencia: `break` corta el lazo entero. `continue` corta solo la vuelta actual.
- Cuidado: poner el incremento ANTES del continue, si no, lazo infinito.
-->

---

<!-- _header: 'Patrón: Validación' -->

# Validar entrada del usuario

```python
while True:
    edad = int(input("Tu edad (0-120): "))
    
    if 0 < edad < 120:
        break  # Edad válida, salir
    
    print("Edad inválida. Intenta de nuevo")

print(f"Edad válida: {edad}")
```

**Repite hasta obtener dato correcto**

<!--
NOTAS DEL ORADOR:
- Es el uso más común de `while` en programas interactivos.
- "No te dejo pasar hasta que me digas la verdad".
-->

---

<!-- _header: 'Patrón: Menú' -->

# Menú interactivo

```python
while True:
    print("\n=== MENÚ ===")
    print("1. Opción A")
    print("2. Opción B")
    print("3. Salir")
    
    opcion = input("Elige (1-3): ")
    
    if opcion == "1":
        print("Ejecutando A...")
    elif opcion == "2":
        print("Ejecutando B...")
    elif opcion == "3":
        print("¡Adiós!")
        break
    else:
        print("Opción inválida")
```

<!--
NOTAS DEL ORADOR:
- Estructura básica de una aplicación de consola.
-->

<!-- _header: 'Patrón: Acumulador' -->

# Sumar números

```python
suma = 0  # Inicializar acumulador
contador = 1

while contador <= 10:
    suma = suma + contador  # Acumular
    contador += 1

print(f"Suma del 1 al 10: {suma}")
# 55
```

**Patrón:** Inicializar + acumular en lazo

<!--
NOTAS DEL ORADOR:
- La variable `suma` va "creciendo".
- Analogía de la alcancía: empezás con 0 y vas metiendo monedas.
-->

<!-- _header: 'Patrón: Contador' -->

# Contar elementos

```python
texto = "Hola Mundo"
vocales = 0
indice = 0

while indice < len(texto):
    if texto[indice].lower() in "aeiou":
        vocales += 1
    indice += 1

print(f"Vocales: {vocales}")  # 4
```

<!--
NOTAS DEL ORADOR:
- Similar al acumulador, pero suma siempre 1.

<!-- _header: 'Patrón: Bandera' -->

# Buscar en secuencia

```python
numeros = [10, 20, 30, 40, 50]
objetivo = 30
encontrado = False  # Bandera
indice = 0

while indice < len(numeros) and not encontrado:
    if numeros[indice] == objetivo:
        encontrado = True
    indice += 1

if encontrado:
    print(f"Encontrado en posición {indice-1}")
else:
    print("No encontrado")
```

<!--
NOTAS DEL ORADOR:
- Uso de variables booleanas para controlar el flujo.
- Optimizamos: si lo encontramos, dejamos de buscar (`and not encontrado`).
-->

<!-- _header: 'while vs for' -->

# ¿Cuándo usar while?

**Usa while cuando:**
* No sabés cuántas veces repetir
* Depende de una condición
* Validaciones y menús
* Lazos controlados por usuario

**Ejemplos:**
* Pedir contraseña hasta que sea correcta
* Menú hasta elegir "Salir"
* Juego hasta que se acabe

<!--
NOTAS DEL ORADOR:
- Indeterminación.
- "Repetí esto hasta que pase X".
-->

---

<!-- _header: 'Ejemplo práctico' -->

# Adivina el número

```python
import random

numero_secreto = random.randint(1, 100)
intentos = 0

while True:
    intento = int(input("Adivina (1-100): "))
    intentos += 1
    
    if intento == numero_secreto:
        print(f"¡Correcto en {intentos} intentos!")
        break
    elif intento < numero_secreto:
        print("Muy bajo")
    else:
        print("Muy alto")
```

<!--
NOTAS DEL ORADOR:
- Ejemplo integrador divertido.
- Combina `random`, `while`, `if/elif`, `input` y contadores.
-->

<!-- _header: 'Errores comunes' -->

# Error 1: No inicializar

**❌ Error:**
```python
while contador < 5:  # contador no existe
    print(contador)
```

**✅ Correcto:**
```python
contador = 0  # Inicializar primero
while contador < 5:
    print(contador)
    contador += 1
```

<!--
NOTAS DEL ORADOR:
- NameError.
-->

---

<!-- _header: 'Errores comunes' -->

# Error 2: Condición siempre True

**❌ Error:**
```python
while True:
    print("Hola")
    # No hay break, no hay salida
```

**✅ Correcto:**
```python
while True:
    respuesta = input("Continuar? (s/n): ")
    if respuesta == 'n':
        break
```

<!--
NOTAS DEL ORADOR:
- Siempre debe haber una "puerta de salida".
-->

# Suma de números positivos

**Crea un programa que:**
1. Pida números al usuario
2. Los sume
3. Se detenga cuando ingrese 0
4. Muestre la suma total

**Pista:** Usa while True y break

<!--
NOTAS DEL ORADOR:
- Tiempo para el ejercicio (5 min).
-->

---

<!-- _header: 'Ejercicio - Solución' -->

# Posible solución

```python
suma = 0

while True:
    numero = float(input("Número (0 para terminar): "))
    
    if numero == 0:
        break
    
    suma += numero

print(f"Suma total: {suma}")
```
