# Imprimir vs Devolver

**Imprimir (print):**
```python
def sumar_print(a, b):
    resultado = a + b
    print(resultado)  # Solo muestra

sumar_print(3, 5)  # Imprime: 8
# No podés usar el resultado
```

**Devolver (return):**
```python
def sumar_return(a, b):
    return a + b  # Devuelve el valor

resultado = sumar_return(3, 5)  # resultado = 8
print(resultado * 2)  # Podés usarlo
```

<!--
NOTAS DEL ORADOR:
- Analogía del Cajero Automático:
    - `print`: El cajero te muestra el saldo en pantalla, pero no te da la plata.
    - `return`: El cajero te entrega los billetes en la mano.
- Con los billetes (return) podés ir a comprar cosas. Con la imagen en pantalla (print) no podés hacer nada más que mirarla.
-->

<!-- _header: 'return' -->

# Palabra clave return

**Sintaxis:**
```python
def funcion():
    # Hacer algo
    return valor  # Devolver resultado
```

**Ejemplo:**
```python
def calcular_cuadrado(numero):
    """Devuelve el cuadrado de un número."""
    return numero ** 2

# Usar el valor devuelto
x = calcular_cuadrado(5)
print(x)  # 25
print(calcular_cuadrado(3) + calcular_cuadrado(4))  # 25
```

<!--
NOTAS DEL ORADOR:
- La palabra clave `return` hace dos cosas:
    1. Define qué valor entrega la función.
    2. Termina inmediatamente la ejecución de la función.
- Mostrar cómo se puede encadenar: calcular el cuadrado de 3 y 4 y sumarlos directamente.
-->

<!-- _header: 'Sin return' -->

# return None implícito

**Si no hay return:**
```python
def saludar(nombre):
    print(f"Hola, {nombre}")
    # No hay return

resultado = saludar("Ana")  # Imprime: Hola, Ana
print(resultado)  # None
```

**Todas las funciones devuelven algo**
* Con return → lo que especifiques
* Sin return → None

<!--
NOTAS DEL ORADOR:
- En Python, si no escribís `return`, Python escribe `return None` por vos al final.
- `None` es un valor real que representa "ausencia de valor".
- Es un error común intentar asignar el resultado de una función que solo imprime: `x = lista.sort()` (que devuelve None).
-->

<!-- _header: 'return detiene' -->

# Return termina la función

```python
def es_mayor_edad(edad):
    """Verifica si es mayor de edad."""
    if edad >= 18:
        return True  # Termina aquí
    return False  # Solo si no entró al if

print(es_mayor_edad(20))  # True
print(es_mayor_edad(15))  # False
```

**Después de return no se ejecuta nada**

<!--
NOTAS DEL ORADOR:
- Esto se llama "Guard Clause" o cláusula de guarda.
- No hace falta `else` porque si entra al `if`, la función termina. Si sigue ejecutando, es porque no entró.
- Es más limpio que tener muchos `if-else` anidados.
-->

# Varios puntos de salida

```python
def clasificar_nota(nota):
    """Clasifica una nota."""
    if nota >= 90:
        return "Excelente"
    elif nota >= 70:
        return "Aprobado"
    elif nota >= 50:
        return "Suficiente"
    else:
        return "Insuficiente"

print(clasificar_nota(85))  # Aprobado
```

**Solo uno se ejecuta**

<!--
NOTAS DEL ORADOR:
- Una función puede tener muchas puertas de salida, pero solo usará una por ejecución.

# Devolver varios valores

**Python permite devolver múltiples:**
```python
def dividir_completo(dividendo, divisor):
    """Devuelve cociente y resto."""
    cociente = dividendo // divisor
    resto = dividendo % divisor
    return cociente, resto

# Recibir ambos valores
c, r = dividir_completo(17, 5)
print(f"Cociente: {c}, Resto: {r}")
# Cociente: 3, Resto: 2
```

**Devuelve tupla: (valor1, valor2)**

<!--
NOTAS DEL ORADOR:
- Técnicamente devuelve UNA tupla.
- Pero gracias al "Unpacking" (desempaquetado) que vimos antes, podemos asignarlo a variables separadas en una línea.
- Muy útil en coordenadas (x, y), operaciones matemáticas (divmod), etc.
-->

<!-- _header: 'Usar resultados' -->

# Guardar y reutilizar

```python
def calcular_promedio(notas):
    """Calcula promedio de notas."""
    return sum(notas) / len(notas)

# Guardar resultado
promedio1 = calcular_promedio([7, 8, 9])
promedio2 = calcular_promedio([6, 7, 8])

# Usar en operaciones
diferencia = promedio1 - promedio2
print(f"Diferencia: {diferencia:.2f}")

# Usar en condiciones
if promedio1 > 7.5:
    print("Buen promedio")
```

<!--
NOTAS DEL ORADOR:
- Este es el verdadero poder. Separamos el cálculo del uso.
- La función no sabe para qué usaremos el promedio (imprimirlo, compararlo, guardarlo en BD).
- Esto aumenta la reusabilidad.
-->

# Funciones llamando funciones

```python
def cuadrado(n):
    return n ** 2

def suma_cuadrados(a, b):
    """Suma de cuadrados de dos números."""
    return cuadrado(a) + cuadrado(b)

resultado = suma_cuadrados(3, 4)
print(resultado)  # 9 + 16 = 25
```

**Construir funciones complejas**

# Validar email

```python
def es_email_valido(email):
    """Verifica si email es válido (simple)."""
    # Reglas básicas
    if "@" not in email:
        return False
    if "." not in email:
        return False
    if len(email) < 5:
        return False
    return True

# Usar
email = "usuario@ejemplo.com"
if es_email_valido(email):
    print("Email válido")
else:
    print("Email inválido")
```

<!--
NOTAS DEL ORADOR:
- Ejemplo de función booleana (predicado).
- Nótese cómo usamos `return False` varias veces para descartar casos inválidos rápido (Fail Fast).

<!-- _header: 'Return vs Print' -->

# Comparación

**Print:**
* Muestra en pantalla
* No se puede reutilizar
* Para debug o salida final

**Return:**
* Devuelve valor
* Se puede guardar y usar
* Para lógica del programa

```python
# ❌ No reutilizable
def sumar_print(a, b):
    print(a + b)

# ✅ Reutilizable
def sumar_return(a, b):
    return a + b
```

<!--
NOTAS DEL ORADOR:
- Tabla resumen.
- Regla general: Las funciones de lógica de negocio deben usar `return`. Solo la capa de interfaz de usuario usa `print`.
-->

# Calcular IMC

**Crea una función que:**
1. Se llame `calcular_imc`
2. Reciba `peso` (kg) y `altura` (m)
3. Calcule IMC = peso / altura²
4. **Devuelva** el resultado
5. Úsala para calcular y mostrar tu IMC

<!--
NOTAS DEL ORADOR:
- Ejercicio práctico.
- Recordar: altura al cuadrado es `altura ** 2`.
- Verificar que los alumnos NO pongan el print dentro de la función de cálculo, sino fuera.
-->

---

<!-- _header: 'Ejercicio - Solución' -->

# Posible solución

```python
def calcular_imc(peso, altura):
    """Calcula Índice de Masa Corporal."""
    imc = peso / (altura ** 2)
    return imc

# Usar la función
mi_peso = 70
mi_altura = 1.75

mi_imc = calcular_imc(mi_peso, mi_altura)
print(f"Tu IMC: {mi_imc:.1f}")

# Clasificar
if mi_imc < 18.5:
    print("Bajo peso")
elif mi_imc < 25:
    print("Peso normal")
else:
    print("Sobrepeso")
```

<!--
NOTAS DEL ORADOR:
- Mostrar cómo usamos `mi_imc` para dos cosas: imprimirlo Y usarlo en el if.
- Si la función solo hiciera print, no podríamos hacer el if después.
