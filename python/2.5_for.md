<!-- _header: 'for vs while' -->

# Diferencias clave

**while:**
* No sabés cuántas veces repetir
* Condición puede cambiar
* Ejemplo: Validar entrada

**for:**
* Sabés cuántas veces o sobre qué iterar
* Recorre secuencias
* Ejemplo: Procesar lista de nombres

<!--
NOTAS DEL ORADOR:
- While es "Indeterminado" (depende de un evento).
- For es "Determinado" (depende de una colección).
- En Python, `for` es extremadamente potente y se usa más que `while`.
-->

---

<!-- _header: 'for básico' -->

# Sintaxis del for

**Estructura:**
```python
for variable in secuencia:
    # Código que se repite
    # para cada elemento
```

**Ejemplo:**
```python
for letra in "Hola":
    print(letra)
```

**Salida:** H, o, l, a (uno por línea)

<!--
NOTAS DEL ORADOR:
- `variable` es un nombre que nosotros elegimos. Se crea automáticamente en cada vuelta.
- `secuencia` puede ser un string, una lista, un rango, etc.
-->

---

<!-- _header: 'range()' -->

# Generar secuencias numéricas

**range(fin):**
```python
for i in range(5):
    print(i)
# 0, 1, 2, 3, 4
```

**range(inicio, fin):**
```python
for i in range(2, 6):
    print(i)
# 2, 3, 4, 5
```

**range(inicio, fin, paso):**
```python
for i in range(0, 10, 2):
    print(i)
# 0, 2, 4, 6, 8
```

<!--
NOTAS DEL ORADOR:
- `range` es una función fundamental.
- Importante: El valor final NO está incluido.
- Es como los intervalos matemáticos semi-abiertos: [inicio, fin).
-->

---

<!-- _header: 'range()' -->

# Detalles importantes

**Empieza en 0 por defecto:**
```python
range(5)  # 0, 1, 2, 3, 4
```

**No incluye el final:**
```python
range(1, 5)  # 1, 2, 3, 4 (no incluye 5)
```

**Paso negativo:**
```python
for i in range(5, 0, -1):
    print(i)
# 5, 4, 3, 2, 1
```

<!--
NOTAS DEL ORADOR:
- El paso negativo permite contar hacia atrás (cuenta regresiva).
- Si `inicio > fin` y el paso es positivo, el lazo no se ejecuta nunca (no hay error, solo no entra).
-->

<!-- _header: 'Iterar listas' -->

# Recorrer listas

```python
frutas = ["manzana", "banana", "naranja"]

for fruta in frutas:
    print(f"Me gusta la {fruta}")
```

**Salida:**
```
Me gusta la manzana
Me gusta la banana
Me gusta la naranja
```

<!--
NOTAS DEL ORADOR:
- Esta es la forma más "limpia" de recorrer una lista.
- No necesitamos índices manuales (como en C o Java).
- Nombramos a la variable en singular (`fruta`) y a la lista en plural (`frutas`).
-->

---

<!-- _header: 'enumerate()' -->

# Obtener índice y valor

**Sin enumerate:**
```python
frutas = ["manzana", "banana", "naranja"]
indice = 0
for fruta in frutas:
    print(f"{indice}: {fruta}")
    indice += 1
```

**Con enumerate:**
```python
for indice, fruta in enumerate(frutas):
    print(f"{indice}: {fruta}")
```

**Más claro y pythónico**

<!--
NOTAS DEL ORADOR:
- ¿Qué pasa si necesito saber en qué posición estoy?
- `enumerate` nos devuelve dos cosas en cada vuelta: la posición y el objeto.
-->

---

<!-- _header: 'enumerate()' -->

# Empezar desde 1

```python
frutas = ["manzana", "banana", "naranja"]

for num, fruta in enumerate(frutas, start=1):
    print(f"{num}. {fruta}")
```

**Salida:**
```
1. manzana
2. banana
3. naranja
```

<!--
NOTAS DEL ORADOR:
- Muy útil para mostrar listas numeradas al usuario.
-->

<!-- _header: 'break y continue' -->

# Control del lazo

**break - salir:**
```python
for i in range(10):
    if i == 5:
        break
    print(i)
# 0, 1, 2, 3, 4
```

**continue - siguiente:**
```python
for i in range(5):
    if i == 2:
        continue
    print(i)
# 0, 1, 3, 4 (salta el 2)
```

<!--
NOTAS DEL ORADOR:
- Funcionan igual que en el `while`.
-->

<!-- _header: 'Patrón: Suma' -->

# Acumular valores

```python
numeros = [10, 20, 30, 40, 50]
suma = 0

for numero in numeros:
    suma += numero

print(f"Suma total: {suma}")  # 150
```

**Patrón:** Inicializar + acumular

<!--
NOTAS DEL ORADOR:
- Recordar inicializar la variable `suma` antes de empezar el `for`.

<!-- _header: 'Patrón: Contar' -->

# Contar elementos

```python
texto = "Hola Mundo"
vocales = 0

for caracter in texto:
    if caracter.lower() in "aeiou":
        vocales += 1

print(f"Vocales: {vocales}")  # 4
```

<!--
NOTAS DEL ORADOR:
- Notar el uso de `.lower()` y el operador `in` para simplificar la comparación.
-->

<!-- _header: 'Patrón: Buscar' -->

# Encontrar elemento

```python
nombres = ["Ana", "Juan", "María", "Pedro"]
buscado = "María"
encontrado = False

for nombre in nombres:
    if nombre == buscado:
        encontrado = True
        break

if encontrado:
    print("Encontrado")
else:
    print("No encontrado")
```

<!--
NOTAS DEL ORADOR:
- El `break` es importante por eficiencia: una vez que lo encontré, ¿para qué seguir mirando los otros 1000 nombres?
-->


**Mejor con comprensión de listas (más adelante)**

<!-- _header: 'Lazos anidados' -->

# Un for dentro de otro

```python
for i in range(3):
    for j in range(3):
        print(f"({i}, {j})", end=" ")
    print()  # Nueva línea
```

**Salida:**
```
(0, 0) (0, 1) (0, 2) 
(1, 0) (1, 1) (1, 2) 
(2, 0) (2, 1) (2, 2) 
```

<!--
NOTAS DEL ORADOR:
- Se usa para grillas, tablas, coordenadas.
- El lazo interno se ejecuta COMPLETAMENTE por cada vuelta del lazo externo.
- En total se ejecutan 3 * 3 = 9 veces el print interno.
-->

<!-- _header: 'Ejemplo: Tabla' -->

# Tabla de multiplicar

```python
numero = 5

for i in range(1, 11):
    resultado = numero * i
    print(f"{numero} x {i} = {resultado}")
```

**Salida:**
```
5 x 1 = 5
5 x 2 = 10
...
5 x 10 = 50
```

<!-- _header: '¿Cuándo usar for?' -->

# for vs while

**Usa for cuando:**
* Conocés la secuencia a recorrer
* Procesás cada elemento de una lista
* Repetís N veces conocidas
* Trabajás con rangos

**Usa while cuando:**
* No sabés cuántas veces
* Depende de condición variable
* Validaciones
* Menús interactivos

# Suma de pares

**Crea un programa que:**
1. Recorra números del 1 al 20
2. Sume solo los pares
3. Muestre el resultado

**Pista:** Usa `numero % 2 == 0` para pares

<!--
NOTAS DEL ORADOR:
- Tiempo para el ejercicio (5 min).
-->

---

<!-- _header: 'Ejercicio - Solución' -->

# Posible solución

```python
suma_pares = 0

for numero in range(1, 21):
    if numero % 2 == 0:
        suma_pares += numero

print(f"Suma de pares (1-20): {suma_pares}")
# Resultado: 110
```
