# Listas y Sets

**Lista:**
* Secuencia ordenada

**Set:**
* Colecci√≥n desordenada

# Listas

**Usa listas cuando:**

# Sets

**Usa sets cuando:**

* B√∫squeda r√°pida

# Contar elementos

**Ambos usan len():**

**len() funciona con todas las estructuras**

<!--
NOTAS DEL ORADOR:
- `len` es polim√≥rfico. Funciona con strings, listas, sets, tuplas, diccionarios.

# Verificar si existe

**Usar in:**
```python

print("python" in tags)  # True
print("ruby" in tags)    # False
```

<!--
NOTAS DEL ORADOR:
