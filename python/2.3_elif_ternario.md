**Con if-else solo hay 2 opciones:**

**¬øY si hay 3, 4, 5 opciones?**

# Normal vs ternario

**Forma normal:**
```python

    estado = "Mayor"

    estado = "Menor"
```

**Forma ternaria:**

**Mismo resultado, m√°s compacto**

# Calculadora de IMC

**Crea un programa que:**
1. Pida peso (kg) y altura (m)
2. Calcule IMC = peso / altura¬≤
3. Clasifique:
   * < 18.5: Bajo peso
   * 18.5-24.9: Normal
   * 25-29.9: Sobrepeso

---

<!-- _header: 'Ejercicio - Soluci√≥n' -->

# Posible soluci√≥n

```python

    print("Bajo peso")

    print("Obesidad")

```

<!--
NOTAS DEL ORADOR:
