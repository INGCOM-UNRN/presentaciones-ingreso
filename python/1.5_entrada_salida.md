# Hablando con el usuario

**Hasta ahora:**
* El programa "habla solo"
* Solo muestra cosas

**Programas útiles necesitan:**
* Pedir información (input)
* Mostrar resultados (output)
* Es como una conversación

<!--
NOTAS DEL ORADOR:
- Modelo IPO: Input - Process - Output.
- Sin input, un programa siempre hace lo mismo.
-->

---

<!-- _header: 'Las dos funciones' -->

# input() y print()

**print() - Salida:**
* Muestra información en pantalla
* Le "habla" al usuario

**input() - Entrada:**
* Pide datos al usuario
* Espera que escriba algo
* Retorna lo que escribió

<!--
NOTAS DEL ORADOR:
- `input()` detiene la ejecución del programa hasta que el usuario presiona Enter.
- Esto es "bloqueante".
-->

# <!-- fit --> print()
## Mostrar información

<!--
NOTAS DEL ORADOR:

<!-- _header: 'print() básico' -->

# Usos básicos

**Texto simple:**
```python
print("Hola Mundo")
```

**Variables:**
```python
nombre = "Ana"
print(nombre)
```

**Múltiples valores:**
```python
edad = 20
print("Nombre:", nombre, "Edad:", edad)
```

<!--
NOTAS DEL ORADOR:
- Notar que la coma `,` en print agrega un espacio automáticamente.
-->

---

<!-- _header: 'F-strings' -->

# La mejor forma: f-strings

**Sintaxis:** poner `f` antes de las comillas

```python
nombre = "Carlos"
edad = 25

print(f"Mi nombre es {nombre}")
print(f"Tengo {edad} años")
print(f"Próximo año: {edad + 1} años")
```

**Salida:**
```
Mi nombre es Carlos
Tengo 25 años
Próximo año: 26 años
```

<!--
NOTAS DEL ORADOR:
- Introducida en Python 3.6.
- Es la forma moderna y recomendada.
- "f" viene de "format".
- Las llaves `{}` son marcadores de posición para código Python.
-->

---

<!-- _header: 'F-strings' -->

# Expresiones dentro

**Podés poner cualquier expresión:**
```python
a = 10
b = 5

print(f"{a} + {b} = {a + b}")
print(f"{a} * {b} = {a * b}")
print(f"Mayor: {max(a, b)}")
```

**Salida:**
```
10 + 5 = 15
10 * 5 = 50
Mayor: 10
```

<!--
NOTAS DEL ORADOR:
- No solo variables, cualquier código válido va dentro de `{}`.
- Útil para cálculos rápidos en el mensaje.
-->

---

<!-- _header: 'Formateo' -->

# Formateo de números

**Decimales:**
```python
precio = 19.99
print(f"Precio: ${precio:.2f}")
# Salida: Precio: $19.99
```

**Sin decimales:**
```python
print(f"Precio: ${precio:.0f}")
# Salida: Precio: $20
```

**`.2f` = 2 decimales**

<!--
NOTAS DEL ORADOR:
- Crucial para dinero o medidas.
- `.2f` significa "Fixed point with 2 decimals".
-->

---

<!-- _header: 'Separadores' -->

# Personalizar print()

**Cambiar separador:**
```python
print("A", "B", "C", sep="-")
# Salida: A-B-C
```

**Cambiar final:**
```python
print("Hola", end=" ")
print("Mundo")
# Salida: Hola Mundo (misma línea)
```

<!--
NOTAS DEL ORADOR:
- `end` por defecto es `\n` (salto de línea). Cambiarlo permite imprimir en la misma línea.
-->

<!--

<!-- _header: 'input() básico' -->

# Leer datos

**Sintaxis:**
```python
variable = input("Mensaje: ")
```

**Ejemplo:**
```python
nombre = input("¿Cómo te llamás? ")
print(f"Hola {nombre}!")
```

**Ejecución:**
```
¿Cómo te llamás? Ana
Hola Ana!
```

<!--
NOTAS DEL ORADOR:
- El texto dentro de `input()` es el "prompt" o mensaje al usuario.
- Siempre hay que asignar el resultado a una variable, si no, se pierde.
-->

---

<!-- _header: 'Importante' -->

# input() siempre retorna texto

**Siempre es tipo str:**
```python
edad = input("Tu edad: ")
print(type(edad))  # <class 'str'>
```

**Aunque escribas un número:**
```python
# Usuario escribe: 18
edad = input("Edad: ")  # edad = "18" (texto)
```

**Para números, hay que convertir**

<!--
NOTAS DEL ORADOR:
- Este es el error #1 con input.
- "18" no es igual a 18.
- No se puede sumar "18" + 1.
-->

---

<!-- _header: 'Conversión' -->

# Convertir a número

**Para usar como número:**
```python
edad_texto = input("Tu edad: ")
edad = int(edad_texto)  # Convertir a int

# O en una línea:
edad = int(input("Tu edad: "))
```

**Para decimales:**
```python
altura = float(input("Tu altura: "))
```

<!--
NOTAS DEL ORADOR:
- Casting (casteo).
- `int()` transforma texto a entero.
- `float()` transforma texto a decimal.
- Anidar funciones: `int(input(...))` es muy común.
-->

---

<!-- _header: 'Ejemplo completo' -->

# Programa interactivo

```python
# Pedir datos
nombre = input("¿Cómo te llamás? ")
edad = int(input("¿Cuántos años tenés? "))
altura = float(input("¿Cuánto medís (m)? "))

# Mostrar información
print(f"\n=== TUS DATOS ===")
print(f"Nombre: {nombre}")
print(f"Edad: {edad} años")
print(f"Altura: {altura:.2f}m")
```

<!--
NOTAS DEL ORADOR:
- Ejemplo integrador.
- Notar el `\n` para dejar un renglón.
-->

---

<!-- _header: 'Calculadora' -->

# Ejemplo: Suma de dos números

```python
print("=== CALCULADORA ===")

num1 = float(input("Primer número: "))
num2 = float(input("Segundo número: "))

suma = num1 + num2
resta = num1 - num2
producto = num1 * num2

print(f"\nResultados:")
print(f"{num1} + {num2} = {suma}")
print(f"{num1} - {num2} = {resta}")
print(f"{num1} * {num2} = {producto}")
```

<!--
NOTAS DEL ORADOR:
- Si no usáramos float(), la suma concatenaría el texto ("10" + "5" = "105").
-->

<!-- _header: 'Errores comunes' -->

# Errores típicos

**Error 1: No convertir a número**
```python
edad = input("Edad: ")  # "18" (texto)
edad = edad + 1  # TypeError
```

**Solución:**
```python
edad = int(input("Edad: "))  # 18 (número)
edad = edad + 1  # 19 ✓
```

<!--
NOTAS DEL ORADOR:
- TypeError: can only concatenate str (not "int") to str.
-->

---

<!-- _header: 'Errores comunes' -->

# Más errores

**Error 2: Convertir texto no numérico**
```python
edad = int(input("Edad: "))
# Usuario escribe: "abc"
# ValueError: invalid literal
```

**Solución:** Validar entrada (lo veremos después)

# Tu turno

**Crea un programa que:**
1. Pida el nombre del usuario
2. Pida su edad
3. Calcule en qué año cumplirá 100 años
4. Muestre el resultado

**Pista:** año_actual + (100 - edad)

<!--
NOTAS DEL ORADOR:
- Ejercicio clásico.
-->

---

<!-- _header: 'Ejercicio - Solución' -->

# Posible solución

```python
# Pedir datos
nombre = input("¿Cómo te llamás? ")
edad = int(input("¿Cuántos años tenés? "))

# Calcular
años_faltan = 100 - edad
año_actual = 2026  # o usar datetime
año_cien = año_actual + años_faltan

# Mostrar resultado
print(f"\n{nombre}, cumplirás 100 años")
print(f"en el año {año_cien}")
```

<!--
NOTAS DEL ORADOR:
- Separar entrada, proceso y salida.
