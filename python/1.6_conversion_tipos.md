<!-- _header: '¿Qué es?' -->

# Cambiar de un tipo a otro

**Conversión de tipos (casting):**
* Transformar un valor de un tipo a otro
* Como un traductor de "idiomas" de datos

**¿Cuándo lo necesitás?**
* input() devuelve texto
* Necesitás número para cálculos
* Concatenar números con texto

<!--
NOTAS DEL ORADOR:
- También conocido como "Casting".
- Python es de tipado fuerte: no mezcla tipos incompatibles automáticamente (como sumar número + texto). Hay que hacerlo explícito.
-->

---

<!-- _header: 'El problema' -->

# input() siempre da texto

**Ejemplo del problema:**
```python
edad = input("Tu edad: ")
# Usuario escribe: 18
# Pero edad = "18" (texto, no número)

edad_futura = edad + 5  # ❌ TypeError
```

**No podés sumar texto con número**

<!--
NOTAS DEL ORADOR:
- Error clásico de principiante.
- Explicar TypeError: can only concatenate str (not "int") to str.
-->

---

<!-- _header: 'La solución' -->

# Convertir a número

**Usar int():**
```python
edad = input("Tu edad: ")  # "18" (texto)
edad = int(edad)           # 18 (número)
edad_futura = edad + 5     # 23 ✓
```

**O en una línea:**
```python
edad = int(input("Tu edad: "))
```

<!--
NOTAS DEL ORADOR:
- El patrón de anidar `int(input(...))` es muy común.
- Ojo: Si el usuario escribe algo que no es un número, esto fallará.
-->

---

<!-- _header: 'Funciones de conversión' -->

# Las 4 funciones principales

**int() - A entero:**
```python
int("42")    # 42
int(3.14)    # 3 (trunca)
int("3.14")  # ❌ ValueError
```

**float() - A decimal:**
```python
float("3.14")  # 3.14
float(42)      # 42.0
float("42")    # 42.0
```

<!--
NOTAS DEL ORADOR:
- Notar que `int()` trunca, no redondea. `int(3.99)` es 3.
- `int("3.14")` falla porque el string tiene un punto que no es válido en un entero.
-->

---

<!-- _header: 'Funciones de conversión' -->

# Más conversiones

**str() - A texto:**
```python
str(42)      # "42"
str(3.14)    # "3.14"
str(True)    # "True"
```

**bool() - A booleano:**
```python
bool(1)      # True
bool(0)      # False
bool("hola") # True
bool("")     # False
```

<!--
NOTAS DEL ORADOR:
- `str()` funciona con casi todo. Es la forma segura de convertir algo para imprimirlo concatenado.
- `bool()` sigue reglas específicas sobre qué es "falso" o "verdadero".
-->

---

<!-- _header: 'String a número' -->

# De texto a número

**A entero:**
```python
numero_str = "42"
numero_int = int(numero_str)
print(numero_int + 8)  # 50
```

**A decimal:**
```python
precio_str = "19.99"
precio_float = float(precio_str)
print(precio_float * 2)  # 39.98
```

<!--
NOTAS DEL ORADOR:
- Útil al leer archivos de texto o bases de datos donde todo viene como string.
-->

---

<!-- _header: 'Número a string' -->

# De número a texto

**Para concatenar:**
```python
edad = 25
edad_str = str(edad)
mensaje = "Tengo " + edad_str + " años"
print(mensaje)  # Tengo 25 años
```

**Con f-string (automático):**
```python
edad = 25
print(f"Tengo {edad} años")  # No hace falta convertir
```

<!--
NOTAS DEL ORADOR:
- Las f-strings hacen `str()` implícitamente por nosotros.
- Es otra razón para preferir f-strings sobre concatenación con `+`.
-->

---

<!-- _header: 'Float a int' -->

# Decimal a entero

**Trunca decimales (no redondea):**
```python
altura = 1.85
altura_int = int(altura)
print(altura_int)  # 1 (no 2)

precio = 9.99
precio_int = int(precio)
print(precio_int)  # 9
```

**Para redondear usa round():**
```python
round(1.85)  # 2
```

<!--
NOTAS DEL ORADOR:
- Importante distinción: `int()` corta la parte decimal. `round()` usa reglas de redondeo.
-->

---

<!-- _header: 'A booleano' -->

# Conversión a bool

**Valores "falsos" (Falsy):**
```python
bool(0)      # False
bool(0.0)    # False
bool("")     # False (string vacío)
bool([])     # False (lista vacía)
bool(None)   # False
```

**Todo lo demás es True:**
```python
bool(1)      # True
bool("hola") # True
bool(3.14)   # True
```

<!--
NOTAS DEL ORADOR:
- Concepto de "Truthy" y "Falsy".
- Muy pythónico: `if nombre:` verifica si la variable nombre no está vacía.
-->

<!-- _header: 'Ejemplo práctico' -->

# Calculadora de edad futura

```python
# Pedir edad actual
edad_str = input("¿Cuántos años tenés? ")
edad = int(edad_str)  # Convertir a número

# Calcular
años_futuros = int(input("¿Cuántos años al futuro? "))
edad_futura = edad + años_futuros

# Mostrar resultado
print(f"En {años_futuros} años tendrás")
print(f"{edad_futura} años")
```

<!--
NOTAS DEL ORADOR:
- Ejemplo integrador simple.
-->

<!-- _header: 'Errores comunes' -->

# Error 1: Texto no numérico

**Problema:**
```python
edad = int("abc")  # ❌ ValueError
```

**No se puede convertir texto no numérico**

**Solución:**
* Validar entrada
* Usar try-except (lo veremos después)

<!--
NOTAS DEL ORADOR:
- Python no puede adivinar qué número es "abc".
-->

---

<!-- _header: 'Errores comunes' -->

# Error 2: Decimal con int()

**Problema:**
```python
numero = int("3.14")  # ❌ ValueError
```

**int() no acepta decimales en texto**

**Solución:**
```python
# Primero a float, después a int
numero = int(float("3.14"))  # 3 ✓
```

<!--
NOTAS DEL ORADOR:
- `int(3.14)` funciona. `int("3.14")` falla.
- La diferencia es que el string "3.14" contiene un carácter '.' que `int()` no sabe procesar. `float()` sí sabe.
-->

---

<!-- _header: 'Errores comunes' -->

# Error 3: Concatenar sin convertir

**Problema:**
```python
edad = 18
mensaje = "Tengo " + edad + " años"
# ❌ TypeError
```

**No se puede concatenar str con int**

**Soluciones:**
```python
# Opción 1: convertir
mensaje = "Tengo " + str(edad) + " años"

# Opción 2: f-string (mejor)
mensaje = f"Tengo {edad} años"
```

<!--
NOTAS DEL ORADOR:
- Reiterar que f-strings es la solución preferida.
-->

# Programa: Conversor de temperatura

**Crea un programa que:**
1. Pida temperatura en Celsius
2. Convierta a Fahrenheit
3. Muestre el resultado

**Fórmula:** F = C × 9/5 + 32

**Pista:** Usa `float(input())` para decimales

<!--
NOTAS DEL ORADOR:
- Ejercicio para afianzar `float()`.
-->

---

<!-- _header: 'Ejercicio 2 - Solución' -->

# Posible solución

```python
print("=== CONVERSOR DE TEMPERATURA ===")

# Pedir temperatura en Celsius
celsius = float(input("Temperatura en °C: "))

# Convertir a Fahrenheit
fahrenheit = celsius * 9/5 + 32

# Mostrar resultado
print(f"{celsius}°C = {fahrenheit:.1f}°F")
```

<!--
NOTAS DEL ORADOR:
- Uso de `:.1f` para mostrar un decimal prolijo.
-->
