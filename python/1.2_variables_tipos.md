<!-- _header: '¿Qué es una variable?' -->

# Caja con etiqueta

**Analogía:**
* Variable = caja con etiqueta
* Guarda información
* Podés cambiar el contenido
* La etiqueta identifica la caja

**En programación:**
* Nombre que guarda un valor
* Espacio en memoria
* Valor puede cambiar

<!--
NOTAS DEL ORADOR:
- La analogía de la caja es la más clásica y efectiva.
- Imaginen una mudanza. Tenés una caja que dice "Libros". Adentro hay libros. Podés sacar los libros y poner juguetes. La caja sigue diciendo "Libros" (aunque ahora sería confuso).
- En Python, técnicamente las variables son "etiquetas" que apuntan a objetos en memoria, pero para principiantes, la "caja" funciona perfecto.
-->

---

<!-- _header: 'Crear variables' -->

# Sintaxis simple

**En Python:**
```python
nombre = valor
```

**Ejemplos:**
```python
edad = 18
nombre = "Ana"
altura = 1.65
es_estudiante = True
```

**El símbolo `=` asigna un valor**

<!--
NOTAS DEL ORADOR:
- ¡Atención! El símbolo `=` NO es "es igual a" (matemática).
- En programación, `=` es ASIGNACIÓN. Significa "Guardar lo de la derecha en la variable de la izquierda".
- Se lee: "Asignar 18 a la variable edad".
-->

---

<!-- _header: 'Ejemplo interactivo' -->

# Crear y mostrar

```python
# Creamos variables
edad = 18
nombre = "Ana"
altura = 1.65
es_estudiante = True

# Mostramos los valores
print(f"Nombre: {nombre}")
print(f"Edad: {edad}")
print(f"Altura: {altura}m")
print(f"¿Es estudiante?: {es_estudiante}")
```

<!--
NOTAS DEL ORADOR:
- Aquí introducimos sutilmente las f-strings (`f"..."`).
- No hace falta explicar f-strings a fondo todavía, solo decir que es una forma mágica de meter variables dentro de texto.
-->

<!-- _header: 'Nomenclatura' -->

# Nombres descriptivos

**Buenos nombres (snake_case):**
* `edad_usuario`
* `precio_total`
* `cantidad_productos`

**Malos nombres:**
* `x`, `a`, `dato` (poco descriptivos)
* `PrecioTotal` (no es snake_case)
* `2edad` (comienza con número)

<!--
NOTAS DEL ORADOR:
- "Solo hay dos cosas difíciles en programación: invalidación de caché y nombrar cosas".
- Un buen nombre de variable cuenta una historia. `d` no dice nada. `dias_transcurridos` dice todo.
- Convención Python: `snake_case` (minúsculas separadas por guiones bajos).
-->

---

<!-- _header: 'Reglas' -->

# Reglas para nombres

**Deben:**
1. Comenzar con letra o `_`
2. Contener letras, números y `_`
3. No tener espacios
4. No ser palabras reservadas

**Sensibles a mayúsculas:**
* `edad` ≠ `Edad` ≠ `EDAD`

<!--
NOTAS DEL ORADOR:
- Palabras reservadas: `if`, `while`, `def`, `class`, etc. Python no te deja usarlas.
- Case sensitivity: Es un error común. Escribir `Edad = 10` y luego tratar de imprimir `edad`.
-->

---

<!-- _header: 'Ejemplos' -->

# Válidos e inválidos

**✅ Válido:**
```python
nombre = "Carlos"
edad_2023 = 20
_valor_privado = 100
```

**❌ Inválido:**
```python
2nombre = "Error"      # Empieza con número
mi-variable = 5        # Tiene guión medio (resta)
for = 10               # Palabra reservada
```

<!--
NOTAS DEL ORADOR:
- El guión medio `-` se interpreta como resta. Por eso usamos guión bajo `_`.
-->

---

<!-- _header: 'Reasignación' -->

# Cambiar valores

**Podés cambiar el valor:**
```python
contador = 0
print(contador)  # 0

contador = 5
print(contador)  # 5

contador = contador + 1
print(contador)  # 6
```

**Las variables son cajas, el contenido cambia**

<!--
NOTAS DEL ORADOR:
- Variable significa que VARÍA.
- `contador = contador + 1` es una expresión que explota la cabeza de los matemáticos, pero tiene todo el sentido para los programadores.
- "Tomá el valor actual de contador, sumale 1, y guardalo de nuevo en contador".
-->

---

<!-- _header: 'Reasignación' -->

# Experimento interactivo

```python
contador = 0
print(f"Inicial: {contador}")

contador = 5
print(f"Después de 5: {contador}")

contador = contador + 1
print(f"Incremento: {contador}")

contador = contador * 2
print(f"Duplicado: {contador}")
```

---

<!-- _header: 'Inicialización' -->

# Siempre inicializar

**❌ Error común:**
```python
print(resultado)
# NameError: name 'resultado' is not defined
```

**✅ Correcto:**
```python
resultado = 0
print(resultado)  # 0
```

**Regla: Variables antes de usarlas**

<!--
NOTAS DEL ORADOR:
- No podés abrir una caja que no compraste.
- NameError es uno de los errores más frecuentes. Significa "No sé de qué me estás hablando".
-->

<!-- _header: '¿Por qué tipos?' -->

# Diferentes información

**En el mundo real:**
* Números para contar
* Palabras para hablar
* Sí/No para decidir

**En Python:**
* No es lo mismo `5 + 3`
* Que `"Hola" + "Mundo"`
* Python necesita saber el tipo

<!--
NOTAS DEL ORADOR:
- La operación `+` hace cosas distintas según el tipo (polimorfismo).
- Suma números. Concatena texto.
-->

---

<!-- _header: 'Tipos básicos' -->

# Los 4 tipos fundamentales

**1. int (entero):**
* Números sin decimales
* Ejemplo: `18`, `-5`, `1000`

**2. float (decimal):**
* Números con decimales
* Ejemplo: `3.14`, `1.65`, `-0.5`

**3. str (cadena):**
* Texto entre comillas
* Ejemplo: `"Ana"`, `'Hola'`

**4. bool (booleano):**
* Verdadero o Falso
* Ejemplo: `True`, `False`

<!--
NOTAS DEL ORADOR:
- Estos son los átomos de la programación.
- Todo lo demás (listas, objetos, bases de datos) se construye a partir de estos tipos básicos.
-->

---

<!-- _header: 'int - Enteros' -->

# Números enteros

**Ejemplos de la vida real:**
* Tu edad: 18 años
* Días de la semana: 7
* Temperatura: -5°C
* Población: 1.000.000

**En Python:**
```python
edad = 18
temperatura = -5
año = 2023
poblacion = 1000000
```

<!--
NOTAS DEL ORADOR:
- Python maneja enteros arbitrariamente grandes. Podés calcular 2 elevado a la 1000 y te da el número exacto.
-->

<!-- _header: 'float - Decimales' -->

# Números con decimales

**Ejemplos de la vida real:**
* Altura: 1.65 metros
* Precio: 19.99 pesos
* Pi: 3.14159
* Temperatura: 36.5°C

**En Python:**
```python
altura = 1.65
```

<!--
NOTAS DEL ORADOR:
- Se usa PUNTO para decimales, no coma (estándar inglés).
- Cuidado con la precisión de los flotantes (0.1 + 0.2 != 0.3), pero eso es tema avanzado.
-->

**Siempre entre comillas**

<!-- _header: 'bool - Booleanos' -->

# Verdadero o Falso

**Solo dos valores:**
* `True` (verdadero)
* `False` (falso)

**Ejemplos:**
```python
es_estudiante = True
tiene_descuento = False
mayor_de_edad = True
```

**Para decisiones**

<!--
NOTAS DEL ORADOR:
- George Boole.
- Fundamental para la lógica de control (if/else) que veremos luego.
- Importante: Primera letra Mayúscula (`True`, no `true`).
-->

<!-- _header: 'Verificar tipo' -->

# Función type()

**Para saber el tipo:**
```python
edad = 18
print(type(edad))  # <class 'int'>

altura = 1.65
print(type(altura))  # <class 'float'>

nombre = "Ana"
print(type(nombre))  # <class 'str'>

activo = True
print(type(activo))  # <class 'bool'>
```

<!--
NOTAS DEL ORADOR:
- Herramienta de debugging útil. "No sé qué es esto, preguntémosle a Python".
