# Python usa espacios

**A diferencia de otros lenguajes:**
* No usa llaves `{}`
* Usa espacios para bloques
* La indentaci√≥n ES parte del c√≥digo

**Regla de oro:**
* 4 espacios por nivel
* Ser consistente
* Los editores ayudan

<!--
NOTAS DEL ORADOR:
- Tab vs Espacios: La guerra santa. En Python, PEP 8 dice 4 espacios.
- Jupyter y VS Code convierten Tab a 4 espacios autom√°ticamente.

**‚ùå Error: Inconsistente**

  print("Esto")    # 2 espacios

# if vs if-else

**Solo if:**
```python

    print("¬°Vamos al parque!")
# Si no es soleado, no pasa nada
```

**Con if-else:**
```python

    print("¬°Vamos al parque!")

    print("Nos quedamos en casa")
# Siempre hacemos algo
```

<!--
NOTAS DEL ORADOR:
- `if` simple: Acci√≥n opcional.
- `if-else`: Decisi√≥n obligatoria entre alternativas.
-->

<!-- _header: 'Errores comunes' -->

# Error 1: Olvidar los dos puntos

**‚ùå Incorrecto:**
```python

    print("Mayor")  # SyntaxError
```

**‚úÖ Correcto:**
```python

    print("Mayor")
```

**Los `:` son obligatorios**

<!--
NOTAS DEL ORADOR:
- Es el error de sintaxis #1.
-->

---

<!-- _header: 'Errores comunes' -->

# Error 2: Usar = en vez de ==

**‚ùå Incorrecto:**
```python

    print("Tiene 18")
```

**‚úÖ Correcto:**
```python

    print("Tiene 18")
```

**`=` asigna, `==` compara**

<!--
NOTAS DEL ORADOR:
- Error #2. Muy dif√≠cil de ver al principio.
-->
